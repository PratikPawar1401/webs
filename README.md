# Paradise Nursery Shopping Application

Paradise Nursery is a React + Redux shopping cart application for an online
plant store. Users can browse categorized houseplants, add items to the cart,
adjust quantities, and view dynamic totals.

## Features

- Landing page with company name and a Get Started button.
- Product listing with three categories and six unique plants per category.
- Add to Cart functionality with disabled state after adding from listing.
- Shared navbar across Product and Cart pages with Home, Plants, and Cart links.
- Dynamic cart icon count that updates with quantity changes.
- Cart page with thumbnail, name, unit price, line total, and quantity controls.
- Delete button for each cart item.
- Checkout button showing a Coming Soon message.
- Continue Shopping button returning to the product listing page.

## Tech Stack

- React
- Redux Toolkit
- React Redux
- React Router DOM
- Vite

## Run Locally

1. Install dependencies:

	npm install

2. Start the development server:

	npm run dev

3. Build for production:

	npm run build
